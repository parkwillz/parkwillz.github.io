MIS Field Project

My team and I created a web application for the MIS Department. This application serves as a location where clients and companies can come in, create an account, and submit project ideas for the field project class. It comes with a fully functional login page, project dashboard where clients can see the status of projects they submit, and an admin dashboard where administrators can view client contact information, assign them to a company, and view submitted projects to approve, review or reject. The site also comes with a dark mode! Tools used: ASP .Net, Bootstrap, Microsoft SQL Server, HTML, CSS, C#, Javascript.
